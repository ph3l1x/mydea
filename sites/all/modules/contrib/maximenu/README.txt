Installation:
===========================================================================================================
1) You can Download/Drush or use Update Manager
2) By default, only users with administrative privileges can configure Maximenu


Configurations:
===========================================================================================================
1) List             /admin/structure/maximenu/list
2) Maxedit          /admin/structure/maximenu/maxedit
3) Settings         /admin/structure/maximenu/settings
4) Advanced Editing /admin/structure/maximenu/advanced-editing
5) Import           /admin/structure/maximenu/import


Styles:
===========================================================================================================
1) Accordion
2) Drop
3) Modal
4) Roundabout
5) Tab


Skin:
===========================================================================================================
1) Bubble
2) None
3) Simple


Effects:
===========================================================================================================
1) Dock
2) Jiggly
3) Lavalamp
4) Slideup


Others:
===========================================================================================================
1) Active
2) Autoscroll
3) Longmenu
